The model takes satellite imagery and forest field measurements as input and produces estimates of above-ground biomass across forest landscapes. No specialised hardware is required. The key limitation is geographic bias, it was trained on Senegalese forest conditions and will need retraining with local field data before being applied elsewhere. The tool and datasets are openly licensed for replication and adaptation, but users are asked to credit the original work and link any derivative products back to the source, this helps the community track how the tool develops over time.
