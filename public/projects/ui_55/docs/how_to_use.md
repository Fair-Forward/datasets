## Using the iMaster-DocuCam Landslide Monitoring System

The iMaster-DocuCam is a hardware-software system by Hesotech for long-term, high-resolution visual documentation of landslides, earthfalls, rockfalls, and earth movements across large areas.

### How It Works

The system consists of a pan-tilt-zoom (PTZ) camera and a mini-PC housed in an electronics cabinet. It captures images fully automatically at predefined intervals (e.g. every two hours), achieving a resolution of up to 1 cm per cell at 500 m distance. Data is transmitted via local WiFi or LTE/3G/4G networks and stored in a central database alongside complementary sensor readings.

### Accessing Data Remotely

All captured data can be viewed through a standard web browser (Chrome or Firefox) from any location at any time. No proprietary software is required.

### Analysis Tools

The system includes built-in analysis features:

- **Virtual markers** -- place measurement points on images without needing physical markers in the field.
- **Ruler/measurement function** -- analyse surface changes over time.
- **Time-lapse video** -- generate videos from the image series to visualise movement patterns.
- **Annotation tools** -- add text, arrows, shapes, or pixelation to images.
- **Multi-sensor correlation** -- combine camera data with sensor readings for cause-effect analysis.

### Optional Sensors

The system can be extended with:

- Local weather station (temperature, humidity, rainfall).
- GPS data collection.
- Custom sensor interfaces for additional environmental monitoring.
- Generator or solar panel power supply for off-grid deployment.
